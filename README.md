# Matching
Мэтчинг товаров
Для работы над  задачей по разработке алгоритма для мэтчинга товаров была применена библиотека Faiss, позволяющая осуществить быстрый поиск ближайших соседей и кластеризацию в векторном пространстве. Высокая скорость поиска позволяет работать с очень большими данными.
Для решения задачи были подобраны следующие параметры - 25 кластеров и параметр nprobe, равный трем, что позволило посетить три кластера при операции поиска.
Результат был оценен показателем accuracy@5, он получился более 0,6.
В связи с этим, данный алгоритм можно считать действенным для мэтчинга товаров.
